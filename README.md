# Operating System Lab Assignment 3 – Page Replacement Algorithms

This repository contains my Operating System Lab Assignment 3, which focuses on the comprehensive implementation and analysis of classical page replacement algorithms.

## 📘 Assignment Overview

The assignment demonstrates and compares the following page replacement algorithms:

* FIFO (First-In, First-Out)
* LRU (Least Recently Used)
* Optimal Page Replacement
* MRU (Most Recently Used)
* Second Chance (Clock Algorithm)

Each algorithm is simulated using a common reference string to analyze performance based on page faults.

## 🎯 Objectives

* Understand virtual memory management
* Compare different page replacement strategies
* Analyze page fault performance
* Learn practical OS memory concepts

## 📊 Performance Comparison

The assignment includes:

* Step-by-step simulation
* Page fault count
* Algorithm comparison table
* Final analysis and conclusion
